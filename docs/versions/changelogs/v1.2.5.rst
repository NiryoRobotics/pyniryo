New in version 1.2.5
================================

|:sparkles:| Features
-----------------------------------------------------------
 * New functions available:

  * :py:meth:`~pyniryo.api.tcp_client.NiryoRobot.set_home_pose` to set the home pose.
  * :py:meth:`~pyniryo.api.tcp_client.NiryoRobot.reset_home_pose` to reset the home pose.
  * :py:meth:`~pyniryo.api.tcp_client.NiryoRobot.get_home_pose` to get the home pose.

* :py:meth:`~pyniryo.api.tcp_client.NiryoRobot.move_to_home_pose` Replaced hard-coded logic with a service call and now operational on Ned3Pro.
* :py:meth:`~pyniryo.api.tcp_client.NiryoRobot.move` Now provides feedback for out-of-range poses and notifies the client about the problematic joint.
* :py:meth:`~pyniryo.api.tcp_client.NiryoRobot.get_custom_button_state` now returns a boolean not a string.
