Vision APIÔÉÅ

This file presents the different functions and Enums Image Processing available for image processing.

These functions are divided in subsections:

  • Pure Image Processing are used to deal with the thresholding, contours detection, ..

  • Workspaces wise section contains functions to extract workspace and deal with the relative position in the workspace

  • The section Show allows to display images

  • Image Editing contains lot of function which can compress images, add text to image, ‚Ä¶

Pure image processingÔÉÅ

image_functions.threshold_hsv(list_min_hsv, list_max_hsv, reverse_hue=False, use_s_prime=False)ÔÉÅ

Take BGR image (OpenCV imread result) and return thresholded image according to values on HSV (Hue, Saturation, Value) Pixel will worth 1 if a pixel has a value between min_v and max_v for all channels

Parameters:

  • img (numpy.array) ‚Äì image BGR if rgb_space = False

  • list_min_hsv (list[int]) ‚Äì list corresponding to [min_value_H,min_value_S,min_value_V]

  • list_max_hsv (list[int]) ‚Äì list corresponding to [max_value_H,max_value_S,max_value_V]

  • use_s_prime (bool) ‚Äì True if you want to use S channel as S‚Äô = S x V else classic

  • reverse_hue (bool) ‚Äì Useful for Red color cause it is at both extremum

Returns:

threshold image

Return type:

numpy.array
